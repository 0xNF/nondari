About

Nondari was inspired by reading the excellent Regarding Cocktails by Sasha Petraske and his wife, Georgette Moger-Petraske. The book serves as an introduction to cocktails, and teaches a wide array of drink types while adhering to a simple elegance in each recipe.

Each drink in the book also comes with its own hand-crafted graphic representing the drink in a beautifully minimalistic style.

This site aims to recreate the elegance of each drink, while also fixing some minor editing errors from the book.

This site's name, "nondari", is derived from the Japanese term "飲んだり" (nondari), which roughly translated to "while drinking". We hope you use this site while drinking a cocktail, or while thinking about them.

Features

Like the book, each drink has its own unique drawing which can be interacted with to highlight each individual ingredient.

Additionally, we have added a few features to the experience that are only possible on a computer: You can search for all drinks that use a specific ingredient, as well as assemble a personal inventory of your own liquors, juices and other ingredients, which can be used to show you what drinks you're capable of making.
